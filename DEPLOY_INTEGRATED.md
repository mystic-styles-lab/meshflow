# Интеграция Marzban + Proxy Balancer

Мы успешно объединили Marzban и ваш Балансировщик в единую систему.

## Что сделано:
1.  **Единый запуск:** Оба сервиса запускаются одной командой через `docker-compose`.
2.  **Интеграция интерфейса:** В меню Marzban добавлен пункт **"Balancer Dashboard"**, который открывает панель управления балансировщиком.
3.  **Автосборка:** Docker автоматически собирает фронтенд Marzban с нашими изменениями.

## Как запустить (на сервере):

1.  **Загрузите файлы на сервер:**
    Скопируйте всю папку `Marzban-master` на ваш сервер (например, в `/opt/marzban`).

2.  **Запустите:**
    ```bash
    cd /opt/marzban
    docker-compose up -d --build
    ```
    *Флаг `--build` обязателен при первом запуске, чтобы собрать обновленный интерфейс.*

3.  **Использование:**
    *   Откройте Marzban: `http://YOUR_IP:8000/dashboard/`
    *   В верхнем меню (три полоски) выберите **"Balancer Dashboard"**.
    *   Балансировщик откроется в новой вкладке по адресу `http://YOUR_IP:7777`.

## Примечание:
Балансировщик доступен по порту `7777`. Убедитесь, что этот порт открыт в фаерволе сервера.
